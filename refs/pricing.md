# Subscription Pricing Reference

This document outlines the subscription plans and add-ons used as the basis for generating Stripe test data in this project.

## Billing Notes

- All paid plans are billed **per screen**.
- **Annual billing** offers approximately **10% savings** vs. monthly billing.
- Per-screen pricing allows unlimited screens at the listed per-screen rate.
- The Enterprise plan requires a minimum of **25 screens**.

## Subscription Plans

| Plan       | Monthly (per screen) | Annual (per screen) | Notes                |
| ---------- | -------------------- | ------------------- | -------------------- |
| Free       | $0                   | $0                  | Up to 3 screens      |
| Standard   | $10.00               | $9.00               |                      |
| Pro Plus   | $15.00               | $13.50              |                      |
| Engage     | $30.00               | $27.00              |                      |
| Enterprise | $45.00               | $40.50              | Minimum 25 screens   |

### Free
- Monthly: **$0**
- Annual: **$0**
- Up to 3 screens, 1 GB cloud storage, 25 MB file upload limit, 25 basic apps.
- Watermark/logo displayed on screens.

### Standard
- Monthly: **$10.00 / screen**
- Annual: **$9.00 / screen**
- Playlists and content scheduling, 100+ apps and integrations, templates, dynamic feeds, multi-user, unlimited cloud storage, split-screen zones, designer tool, email support.

### Pro Plus
- Monthly: **$15.00 / screen**
- Annual: **$13.50 / screen**
- Everything in Standard, plus advanced roles, folder-level security, content approval workflow, custom fonts, proof-of-play reporting, custom-branded portal, Microsoft 365 and Google Workspace integration, BI dashboards (Power BI, Salesforce, Looker), secured website display, Slack and Teams integration, dynamic data mapping, power scheduling, team workspaces, campaign management, audit logs, SAML SSO, email and phone support.

### Engage
- Monthly: **$30.00 / screen**
- Annual: **$27.00 / screen**
- Everything in Pro Plus, plus interactive kiosk designer, content library kiosks, IoT sensor integration ("lift & learn"), SMS check-in app, QR scan-to-interact, live TV ad overlay, event-based analytics, custom data residency, email/phone/Zoom support.

### Enterprise
- Monthly: **$45.00 / screen** (min. 25 screens)
- Annual: **$40.50 / screen** (min. 25 screens)
- Everything in Engage, plus dedicated CSM, dedicated onboarding/training, large-scale deployment support, priority support, GraphQL API access, PaaS capabilities, mass device provisioning, customized ToS, dedicated account manager.
